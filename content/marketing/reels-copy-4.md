## 릴스 1. 시연형 — "3초 만에 완벽한 선물 찾기"

**컨셉**: 마로 실제 사용 화면을 빠르게 보여주는 시연 릴스
**BGM**: 트렌디한 전환 사운드 (틱톡 "oh no" 계열 또는 경쾌한 비트)
**길이**: 15~20초

### 스크립트

| 시간 | 화면 | 텍스트 오버레이 | 나레이션/자막 |
|------|------|----------------|-------------|
| 0-2초 | 고민하는 표정 클로즈업 | "선물 뭐 사지...?" | (고민 이모지 줌인) |
| 2-4초 | 마로 앱 화면 진입 | "3초면 끝." | 전환 효과 (스와이프) |
| 4-7초 | 관계 선택 탭 | "누구에게?" → 연인 선택 | 탭탭 사운드 |
| 7-9초 | 상황 선택 | "무슨 상황?" → 생일 선택 | |
| 9-11초 | 예산 선택 | "예산은?" → 5만원 선택 | |
| 11-14초 | AI 분석 로딩 애니메이션 | "AI가 분석 중..." | 두근두근 |
| 14-18초 | 결과 3개 카드 등장 | "이유까지 알려줌" | 와 이거 진짜 찐이다 |
| 18-20초 | CTA 화면 | "무료. 로그인 없음." | maro.ai.kr |

### 캡션
```
선물 고르느라 3시간 쓰는 사람 여기 모여봐

AI가 관계랑 상황 분석해서
딱 맞는 선물 3개 골라줌

진짜 3초면 끝나는데 왜 아직도 혼자 고민해?

지금 바로 해봐 (무료임)
maro.ai.kr

#선물추천 #AI선물 #생일선물 #여자친구선물 #남자친구선물 #선물고민 #마로 #선물추천AI #기념일선물 #선물뭐하지
```

## 릴스 2. 비교형 — "AI가 추천한 선물 vs 내가 고른 선물"

**컨셉**: 스플릿 화면으로 AI 추천 vs 본인 선택을 유머러스하게 비교
**BGM**: "Wow" 리액션 사운드 + 비교 밈 BGM
**길이**: 20~25초

### 스크립트

| 시간 | 왼쪽 (내가 고른 선물) | 오른쪽 (AI 추천) | 오버레이 |
|------|---------------------|-----------------|---------|
| 0-3초 | — | — | "여친 생일선물 대결" |
| 3-6초 | 양말 세트 사진 | 브랜드 향수 미니어처 세트 | "상품" |
| 6-9초 | "이거 실용적이잖아" | "향기는 기억을 가장 오래 남겨요" | "선물 이유" |
| 9-12초 | 여친 무표정 | 여친 감동 눈물 | "받는 사람 반응" |
| 12-15초 | "고마워..." (어색) | "이걸 어떻게 알았어?!" (감동) | "리얼 반응" |
| 15-18초 | 관계 위기 자막 | 관계 레벨업 자막 | |
| 18-22초 | — | 마로 앱 화면 | "차이는 AI 한 번 돌렸냐 안 돌렸냐" |
| 22-25초 | — | CTA | maro.ai.kr |

### 캡션
```
나: 양말이 젤 실용적이지
AI: 향기는 기억을 가장 오래 남기는 선물이에요

...AI 말이 맞았다

"왜 이 선물인지" 이유까지 알려주는
관계 맞춤 AI 선물 추천

maro.ai.kr (무료/로그인 불필요)
```

## 릴스 3. 공감 밈 — "선물 고르다 지친 사람 특징"

**컨셉**: "~하는 사람 특징" 밈 포맷으로 공감 유발
**BGM**: 슬로우 피아노 → 반전 비트 드롭
**길이**: 15~20초

### 스크립트

| 시간 | 화면 | 텍스트 |
|------|------|--------|
| 0-2초 | 텍스트 카드 | "선물 고르다 지친 사람 특징" |
| 2-4초 | 쿠팡 검색창에 "선물" 치는 화면 | 1. 검색창에 '선물' 치고 멍 때림 |
| 4-6초 | 장바구니 100개 화면 | 2. 장바구니에 30개 넣고 하나도 안 삼 |
| 6-8초 | 카톡 대화 캡처 (연출) | 3. 친구한테 "이거 어때?" 10번째 물어봄 |
| 8-10초 | 시계 새벽 2시 | 4. 새벽 2시까지 쇼핑몰 돌아다님 |
| 10-12초 | 결국 기프티콘 보내는 화면 | 5. 결국 기프티콘 보냄 (패배) |
| 12-15초 | 반전: 마로 앱 화면 | "이거 쓰면 30초면 끝나는데..." |
| 15-18초 | 결과 화면 + CTA | "관계 맞춤 AI 선물 추천" maro.ai.kr |

### 캡션
```
5번 중에 3개 이상 해당이면 '선물 스트레스 증후군'입니다

증상:
- 검색창 앞에서 30분 멍
- 장바구니만 100개
- "이거 어때?" 무한 반복
- 새벽 2시 쇼핑몰 순례
- 결국 기프티콘 (매번)

처방: maro.ai.kr
AI가 관계+상황+예산 분석해서 딱 3개만 골라줌
30초면 끝. 무료.

#선물고민 #선물스트레스 #공감밈 #선물추천 #기프티콘탈출 #마로 #AI추천 #선물뭐하지 #선물고르기
```

## 릴스 4. 리스트형 — "예산별 선물: 3만원/5만원/10만원"

**컨셉**: 가격대별 추천 선물을 빠르게 나열하는 정보형 릴스
**BGM**: 경쾌한 팝 비트 (정보 전달형)
**길이**: 20~25초

### 스크립트

| 시간 | 화면 | 텍스트 |
|------|------|--------|
| 0-2초 | 타이틀 카드 | "예산별 선물 총정리" |
| 2-5초 | 3만원 배경 | **3만원대** |
| | 상품 1 | 브랜드 핸드크림 세트 |
| | 상품 2 | 프리미엄 디퓨저 |
| | 상품 3 | 감성 무드등 |
| 5-8초 | — | "부담 없이 센스 있게" |
| 8-12초 | 5만원 배경 | **5만원대** |
| | 상품 1 | 브랜드 향수 미니어처 |
| | 상품 2 | 블루투스 스피커 |
| | 상품 3 | 고급 레스토랑 식사권 |
| 12-14초 | — | "정성이 느껴지는 가격대" |
| 14-18초 | 10만원 배경 | **10만원대** |
| | 상품 1 | 스마트워치 |
| | 상품 2 | 프리미엄 향수 정품 |
| | 상품 3 | 호텔 스파 패키지 |
| 18-20초 | — | "특별한 날엔 확실하게" |
| 20-23초 | 마로 앱 화면 | "예산 넣으면 AI가 골라줌" |

```
그래서 AI한테 맡기면 편함
관계+상황+예산 넣으면 딱 3개 골라줌
이유까지 알려줘서 자신 있게 고를 수 있음
```
