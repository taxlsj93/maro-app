## 1) "어버이날 선물 5만원으로 끝내는 법" (실용형, 7장)

[슬라이드 1] 헤드: 어버이날 선물, 5만원으로 끝내는 법 / 서브: 예산 부담 ZERO, 마음 부담도 ZERO 🌷
[슬라이드 2] 헤드: 먼저 이것부터 정해요 / 서브: "드릴 분의 취향"이 예산보다 먼저예요
[슬라이드 3] 헤드: 1. 꽃다발 + 손편지 (2.5만) / 서브: 가장 오래 남는 건 결국 글씨예요
[슬라이드 4] 헤드: 2. 프리미엄 간식 박스 (3만) / 서브: 매일 한 입씩 "딸 생각나네"
[슬라이드 5] 헤드: 3. 홈카페 드립세트 (4만) / 서브: "집에서 쉬는 편" 부모님께 딱
[슬라이드 6] 헤드: 4. 용돈 봉투 + 편지 (5만) / 서브: 가장 솔직하게 좋아하시는 옵션 😂
[슬라이드 7] 헤드: 3분 만에 딱 맞는 조합 찾기 / 서브: maro.ai.kr 에서 관계·예산 넣고 Top 3 받기

[캡션]
5만원이면 부족하다고 느껴지지만, 사실 부모님이 기억하시는 건 금액이 아니더라구요. 매년 고민되는 어버이날, 올해는 예산 5만원 안에서 실용적으로 정리해봤어요. 저장해두고 다음 주말에 천천히 골라보세요 🌷 어떤 조합이 제일 끌려요?

## 2) "시부모님 어버이날 선물, 며느리 1년차의 정답" (고민해결형, 6장)

[슬라이드 1] 헤드: 시부모님 어버이날 선물 / 서브: 며느리 1년차가 찾은 "안 실패하는" 정답 🥹
[슬라이드 2] 헤드: 실패 포인트 3가지 / 서브: 너무 비싸거나 · 너무 싸거나 · 남편이랑 안 맞추거나
[슬라이드 3] 헤드: STEP 1. 남편과 예산부터 맞추기 / 서브: "우리 둘이서 OO만원" 숫자 먼저 정해요
[슬라이드 4] 헤드: STEP 2. 취향 말고 "일상"을 물어보세요 / 서브: 요즘 뭐 하세요? → 거기서 답이 나와요
[슬라이드 5] 헤드: STEP 3. 카드 한 장은 꼭 넣기 / 서브: "어머님, 감사해요" 한 줄이면 충분해요
[슬라이드 6] 헤드: 3분이면 Top 3 나와요 / 서브: maro.ai.kr · 관계 "시부모님" 선택 → 끝

[캡션]
시부모님 선물은 예산도 취향도 남편도 다 변수라 더 어렵더라구요. 1년차 며느리로서 몇 번 실패하면서 알게 된 건 "일상"을 물어보는 게 제일 빠르다는 거였어요. 올해 어버이날에 비슷한 고민 있으신 분들 참고하세요 🌸 여러분의 시부모님 선물 꿀팁도 댓글로 알려주세요!

## 3) "홍삼 말고 이거 어때요?" 5가지 대안 (리스트형, 7장)

[슬라이드 1] 헤드: 홍삼 말고 이거 어때요? / 서브: 어버이날 선물, 매년 똑같지 않게 🎁
[슬라이드 2] 헤드: 왜 홍삼만 돌려막나요 / 서브: 부모님 댁 장식장에 쌓여있는 홍삼박스, 공감? 🫠
[슬라이드 3] 헤드: 1. 수면 케어 세트 / 서브: 요즘 제일 체감되는 "잘 쉬게 해드리기"
[슬라이드 4] 헤드: 2. 손글씨 캘리그라피 액자 / 서브: "마음을 새기는 선물" 그대로
[슬라이드 5] 헤드: 3. 프리미엄 원두·티 구독 / 서브: 한 달 내내 "딸 생각나게" 하는 방식
[슬라이드 6] 헤드: 4. 가벼운 건강검진 예약 / 서브: 말 꺼내기 어려웠다면 이번 기회에
[슬라이드 7] 헤드: 5. 여행 상품권 + 편지 / 서브: 부모님이 제일 오래 기억하는 선물 1위
